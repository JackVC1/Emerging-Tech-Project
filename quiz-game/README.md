# Quiz Game - React Application

A simple quiz game built with React and JSX that uses the REST Countries API to create flag-based quiz questions.

## Project Overview
This is a college assignment project that demonstrates front-end development using HTML, CSS, and JavaScript (React) with external API integration.

## Current Status: Iteration 1
✅ Basic application structure and layout created
- Navbar with Home and Rules tabs
- Footer with copyright information
- Main content area (empty - ready for game content)

## Upcoming Iterations
- **Iteration 2**: Add questions, answering functionality, and rules content
- **Iteration 3**: Integrate REST Countries API for flag data
- **Iteration 4**: Apply custom styling and design
- **Iteration 5**: Add scoring system, scoreboard, and timer

## Installation

1. Install dependencies:
```
npm install
```

2. Start the development server:
```
npm run dev
```

3. The application will automatically open in your browser
